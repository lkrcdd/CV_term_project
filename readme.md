### python
1. __name__ 변수
.py 파일에는 파이썬 인터프리터에 의해 설정되는 __name__이라는 숨겨진 변수가 있다. 
이 변수는 "해당 모듈의 이름"을 가지고 있는 변수로, 현재 .py 파일의 이름을 가지고 있는 변수라는 의미이다. 
ex | abc.py 
-> abc 파일 외부에서의 __name__ = 'abc'
-> abc 파일 내부에서 직접 실행될 때는 '__main__'
